# Line-of-Sight Stealth Puzzle

Line-of-Sight Stealth Puzzle is a browser-based 3D puzzle game where the player must navigate through guarded environments without being seen. The core challenge revolves around understanding and exploiting enemy vision: each guard observes the world through a limited field of view and can only detect the player if there is a clear, unobstructed line of sight.

Players move through levels using basic controls, carefully timing their actions and using walls or obstacles as cover. Each level is designed as a self-contained puzzle that requires planning, spatial awareness, and precise movement. As the game progresses, complexity increases through additional guards, overlapping vision fields, and moving patrol patterns.

The experience emphasizes clarity and logic over reflexes, encouraging players to study guard behavior, anticipate visibility, and solve scenarios methodically. The entire game runs locally in the browser, delivering a lightweight but technically rich 3D stealth experience without relying on any backend systems.
